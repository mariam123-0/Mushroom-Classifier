# 🍄 Mushroom Classifier

A simple and interactive web application that classifies mushrooms as **Edible** or **Poisonous** based on selected features.

---

## 📌 Overview

The **Mushroom Classifier** allows users to input different characteristics of a mushroom through an easy-to-use interface. Based on these inputs, the model predicts whether the mushroom is safe to eat or not.

---

## ⚙️ How It Works

1. The user selects mushroom features from dropdown menus such as:
   - Cap Shape (bell, conical, convex, flat, etc.)
   - Spore Print Color
   - Population
   - Habitat
   - And more...

2. After selecting all features, the user clicks the **Submit** button.

3. The system processes the input using the trained model.

4. The final result is displayed as:
   - 🍽️ **Edible**
   - ☠️ **Poisonous**
